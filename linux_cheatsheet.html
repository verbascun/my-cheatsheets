Shortcuts

Window
Ctrl+Shift+n-> Open new window
Ctrl+Shift+t-> Open new tab
Ctrl+d-> close tab/log out of shell
Navigation
Ctrl+a-> go to the beginning of the line
Ctrl+e-> go to the end of the line
Delete
Ctrl+u-> all to the left of the cursor
Ctrl+k-> all to the right of the cursor
Ctrl+w-> word on the left
Misc
Ctrl+r-> view commands history

Commands

Package manager
apt update-> download pkgs information
apt upgrade-> install upgrades of all pkgs
apt install <pkg name>-> install pkg
apt remove <pkg name>-> remove pkg without configuration
apt purge <pkg name>-> remove pkg with configuration
