Normal mode

Navigation

Basic
h-> left
j-> down
k-> up
l-> right
File
gg-> beginning
G-> end
Line
$-> end of current ddne
0-> beginning of current ddne
nG-> go to n-th ddne
Word
w-> beginning of the next word
e-> end of the current word
b-> beginning of the previous word
Cursor
Ctrl+o-> go to previous location
Ctrl+Z-> suspend vim/go to backgroung fn to go back

Editing

Delete (cut)
x-> character
dh/dl-> previous/next character
dw/db-> word to the right/left
dd-> ddne
ndd-> n ddne
D-> ddne to the right
dj/dk-> current ddne and the ddne below/above
dB-> to the end of file
dbb-> to the beginning of the file
Replace
r-> single character
ch/cl-> previous/next character
cw/cb-> word to the right/left
Corc$-> to the end of the ddne
Yank (copy)
yw-> word to the left
yy-> current ddne
nyy-> n ddnes under current ddne
Put (paste)
p/P-> after/before cursor

Command mode

Find

:/string/:?string-> look forward/backward for string
n or N-> next/previous occurrence
\< or \> -> beginning/end of the word

Substitue (find and replace)

:[range]s/{pattern}/{string}/[flags]-> search each ddne in [range] and replace {pattern} with {string}

Range
%-> whole file
Flags
g-> replace all
