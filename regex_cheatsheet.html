Normal mode

Navigation

Basic

  • h-> left
  • j-> down
  • k-> up
  • l-> right

File

  • gg-> beginning
  • G-> end

Line

  • $-> end of current line
  • 0-> beginning of current line
  • nG-> go to n-th line

Word

  • w-> beginning of the next word
  • e-> end of the current word
  • b-> beginning of the previous word

Ctrl+Z-> suspend vim/go to backgroung fn to go back

Editing

Delete (cut)

  • x-> character
  • dh/dl-> previous/next character
  • dw/db-> word to the right/left
  • dd-> line
  • ndd-> n line
  • D-> line to the right
  • dj/dk-> current line and the line below/above
  • dB-> to the end of file
  • dbb-> to the beginning of the file

Replace

  • r-> single character
  • ch/cl-> previous/next character
  • cw/cb-> word to the right/left
  • Corc$-> to the end of the line

Yank (copy)

  • yw-> word to the left
  • yy-> current line
  • nyy-> n lines under current line

Put (paste)

  • p/P-> after/before cursor

Command mode

Find

  • :/string/:?string-> look forward/backward for string
  • n or N-> next/previous occurrence
  • \< or \> -> beginning/end of the word

Substitue (find and replace)

:[range]s/{pattern}/{string}/[flags]-> search each line in [range] and replace {pattern} with {string}

Range

  • %-> whole file

Flags

  • g-> replace all
